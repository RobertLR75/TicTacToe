Use the `csharp-backend-agent` skill for this session.

Focus on:
- REST APIs with FastEndpoints
- FastEndpoints FluentValidation validators
- FastEndpoints mapper classes
- Business logic in `IRequest<TResponse>` handlers
- Persistence via `SharedLibrary` `IStorageService`
- Storage backends: Redis, MongoDB, PostgreSQL, SQL Server
- Event publishing via RabbitMQ or Azure Service Bus
- Local development and verification with Aspire

When implementing, keep endpoints thin and handlers focused.
