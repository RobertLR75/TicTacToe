# Entity IDs

All entities use `Guid` IDs. IDs are generated by the storage service using `Guid.CreateVersion7()` (time-sortable).

```csharp
// Correct — pass Id = Guid.Empty, storage sets it
var order = new Order { CustomerId = id }; // Id defaults to Guid.Empty
var newId = await storageService.CreateAsync(order, ct);

// Wrong — do not pre-assign IDs
var order = new Order { Id = Guid.NewGuid(), ... };
```

- Always pass entities with `Id = Guid.Empty` — let the storage service generate the ID
- Use `DateTimeOffset` for `CreatedAt` and `UpdatedAt` (set automatically by storage layer, in UTC)
- Never use `Guid.NewGuid()` for entity IDs — use V7 GUIDs via the storage service
